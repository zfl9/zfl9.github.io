锐速破解版（91yun）

发表于 2016-10-23 | 分类于 linux ， 系统优化 | | 阅读次数

锐速和 BBR（魔改版）可以说是 VPS 的标配了，听说 BBR 魔改的加速效果比锐速好一些，但因为 BBR 需要 4.9+ 内核，而我又不太想升级内核，毕竟 VPS 里面还有其他东西，不想再搞的重装系统了。虽然锐速有时候也需要更换内核（CentOS 7.x），但主版本号没变，还是 3.10.0，相对来说安全和稳定一些。

一键脚本

首先感谢 91yun 站长的热心分享，提供了锐速破解版的一键安装脚本，并且一直都在维护更新。
锐速破解版更新贴，重要提示：不支持 OVZ 虚拟技术、不支持 OVZ 虚拟技术、不支持 OVZ 虚拟技术！

一键安装

wget -N --no-check-certificate https://github.com/91yun/serverspeeder/raw/master/serverspeeder.sh && bash serverspeeder.sh

更换内核

CentOS 6.x

# CentOS 6.x 内核更换为：2.6.32-504.3.3.el6.x86_64
rpm -ivh http://soft.91yun.org/ISO/Linux/CentOS/kernel/kernel-firmware-2.6.32-504.3.3.el6.noarch.rpm
rpm -ivh http://soft.91yun.org/ISO/Linux/CentOS/kernel/kernel-2.6.32-504.3.3.el6.x86_64.rpm --force

CentOS 7.x

# CentOS 7.x 内核更换为：3.10.0-229.1.2.el7.x86_64
rpm -ivh http://soft.91yun.org/ISO/Linux/CentOS/kernel/kernel-3.10.0-229.1.2.el7.x86_64.rpm --force

重启后使用 rpm -qa | grep kernel 查看是否安装成功、使用 uname -r 查看当前使用的内核版本，确认是否切换成功。
