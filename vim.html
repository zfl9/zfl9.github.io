vim 配置

修改默认主题
首先将默认主题换成 molokai 主题，因为 molokai 是第三方主题，因此需要先获取主题文件：
curl -4sSkLO https://raw.github.com/tomasr/molokai/master/colors/molokai.vim

为了不与默认的 vim 配置搞混乱，建议在家目录下创建 .vim 配置文件目录，仅对当前用户生效：
mkdir -p ~/.vim/colors/ && mv molokai.vim ~/.vim/colors/

然后创建 ~/.vim/vimrc 配置文件，启用 molokai 主题：
vim ~/.vim/vimrc，添加或修改colo molokai即可生效

安装 vim 插件
为了安装 vim 插件，我们要先安装 vundle 插件管理器：
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

gvim 配置

安装 gvim
链接：https://ftp.nluug.nl/pub/vim/pc/gvim80-069.exe
gvim 安装目录最好不要放在系统盘（C 盘），有些时候可能会出现权限问题；
添加环境变量：vim=gvim安装目录；后面所有修改均以该环境变量为基础，不再重复说明。

安装 msysgit
链接：https://github.com/git-for-windows/git/releases/download/v2.12.2.windows.2/Git-2.12.2.2-64-bit.exe
打开 cmd，输入git --version如果能显示版本号，表示安装成功。

安装 curl
链接：https://dl.uxnr.de/build/curl/curl_winssl_msys2_mingw64_stc/curl-7.53.1/curl-7.53.1.zip
修改 Path 环境变量：添加;curl目录/bin条目即可；
打开 cmd，输入curl --version如果显示版本号，表示安装成功。

安装 ctags
链接：https://raw.github.com/zfl9/vim/master/ctags.exe
因为是绿色可执行文件，因此只需要放到$vim/vim80/目录下。

修改默认主题
获取主题：https://raw.github.com/tomasr/molokai/master/colors/molokai.vim
将主题放到$vim/vim80/colors/molokai.vim；
编辑$vim/_vimrc，启用主题colo molokai。

安装 vim 插件
打开 cmd，git clone https://github.com/VundleVim/Vundle.vim.git；
将 Vundle.vim 目录放至$vim/vimfiles/bundle/Vundle.vim路径；
